Simple Table

Offline-first Progressive Web App(PWA) to manage your school timetable built from scratch using React.js. Start by adding the list of objects and then you can use them to fill up your schedule. The app is made to work on all screen sizes including mobile. Once the user has visited the app, it automatically caches the data so that it works even without internet.
